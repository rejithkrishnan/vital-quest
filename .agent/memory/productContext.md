# Product Context

## Vision
**Vital Quest** is an AI-powered health coach that gamifies fitness. It essentially turns life into an RPG where healthy actions earn XP.

## Core Features (MVP)
1.  **Auth/Profile**: Know the user (Completed).
2.  **Gamification**: XP for actions, Leveling up, Streaks.
3.  **Daily Plan**: AI-generated tasks.
4.  **AI Chat**: Context-aware coaching.

## User Personas
- **The Gamer**: Wants badges, high score, simple progress bars.
- **The Beginner**: Wants clear "do this" instructions.

## Design Philosophy
- **"Google Fit" Aesthetic**: Clean, white/gray backgrounds, vibrant primary colors (Blue #4285F4, Green #34A853).
- **Modern Interactions**: Bottom sheets for selections, haptic feedback on success.
