# Product Context

## Vision
**Vital Quest** is an AI-powered health coach that gamifies fitness. It essentially turns life into an RPG where healthy actions earn XP.

## Core Features
1.  **Goal Intake**: Conversational AI flow to define realistic health goals.
2.  **Daily Roadmap**: AI-generated weekly and daily plans tailored to user preferences.
3.  **Smart Chat**: Context-aware coaching using persistent memory (RAG).
4.  **Meal Analysis**: Photo and text-based nutritional estimation using Gemini.
5.  **Gamification**: XP for actions, Leveling up, and progress tracking.

## User Personas
- **The Gamer**: Wants badges, high score, simple progress bars.
- **The Beginner**: Wants clear "do this" instructions.
- **The Improver**: Wants data-driven insights and long-term health roadmaps.

## Design Philosophy
- **"Google Fit" Aesthetic**: Clean, white/gray backgrounds, vibrant primary colors (Blue #4285F4, Green #34A853).
- **Interactive Feedback**: Haptic feedback, Lottie animations, and bottom-sheet interactions.
- **Clarity**: High contrast, legible typography (Inter/Roboto), and simple data visualizations.
